# Retro Personal Portfolio

A standalone Mahmood 98 desktop portfolio, built from the local restored copy of the `lu5w/personal-website` data.

## What is included

- Old desktop-style shell with boot, login, desktop shortcuts, taskbar, Start menu, and system windows.
- Portfolio, services, tools, FAQ, resume, media, and voice sections.
- Password entry screen with boot progress and welcome flow. The password hint is shown as `mahmood98`.
- Short original generated desktop-style chimes for startup, click, open, close, and error actions. These are not copied Microsoft audio files.
- A Contact Me window with a mail-style form plus Discord and YouTube links.
- Single-window desktop manager: opening one program closes the others, with minimize and close controls.
- Installed app windows for Instagram, LinkedIn, Discord, and YouTube.
- Instagram, YouTube, LinkedIn, and Discord SVG app logos on the desktop and Programs menu.
- Start menu, My Computer explorer, Control Panel, wallpaper switcher, and sound toggle.
- Fixed desktop viewport with page scrolling disabled; content scrolls only inside app windows.
- Turn Off and Restart dialog for the Mahmood 98 system.
- Working minimize, maximize, close, focus, taskbar restore, and internal app scrolling.
- Color-coded window controls: yellow minimize, green maximize, red quit.
- CV-focused Portfolio window with clear skill groups and hobby photos.
- Clean dashboard home with six clear folders instead of messy overlapping content.
- Clickable hobby cards and Discord app details with matching pictures.
- Pinned app dock for YouTube, Discord, LinkedIn, and Instagram.
- All 55 usable images added to Media Library with click-to-enlarge preview.
- Windows can now be dragged mostly off-screen and resized more freely.
- Mahmood.exe home window can now close completely.
- Recycle Bin now uses more online meme images with no descriptions.
- Music app now plays four local generated audio tracks inside the desktop, avoiding YouTube embed errors.
- Mahmood's profile images, project images, hobby images, CV PDF, and language audio clips copied from the restored data folder.
- External links preserved for the Discord server and YouTube channel.

## Notes

The original local source did not include a specific YouTube video embed URL, only the `https://www.youtube.com/@Lu5w` channel link, so the new page includes a YouTube channel panel rather than a video iframe.

Music and the custom/funny recycle bin were intentionally left out for a later pass, as requested. Instagram, LinkedIn, Discord, and YouTube live previews are not installed locally, so their app windows show profile info with launch buttons.

Open `index.html` to start the Mahmood 98 desktop.
